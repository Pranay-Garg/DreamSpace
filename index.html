Privacy Policy

Effective Date: 16/04/2025

Last Updated: 28/05/2025

This Privacy Policy applies to the VR app DreamSpace: The Ceiling Edition, available on the Meta Quest platform.

1. What data we are collecting

DreamSpace: The Ceiling Edition does not collect personal data such as:

  • Names, emails, or contact information
  • Biometric or health data
  • Location or GPS information
  • Voice recordings or images
  • User-generated content or interactions

However, we collect anonymous gameplay statistics using Unity Analytics to improve app performance and design. This includes data such as:

  • Session duration and frequency
  • Feature usage events
  • Crash reports and technical performance metrics

We also temporarily access your Meta User ID to verify the status of in-app purchases. This identifier is used solely for purchase verification and is not stored, tracked, or linked with any personal profile.

2. How we process data

The Meta User ID is accessed only during purchase validation and is not retained beyond that operation.

Unity Analytics provides aggregated, anonymous data that does not identify individual users. This data is used only to understand overall app usage and improve functionality.

Any technical data shared by Meta (e.g., crash logs) is used solely for debugging and app improvement.

3. Why we process data

We process anonymous analytics data in order to:

  • Improve overall app performance and usability
  • Fix bugs and crashes
  • Understand general usage patterns and user behavior within the app

We do not use this data to track, target, or identify individual users.

4. How users can request data deletion

We do not collect or store any personal or user-identifiable data, so there is no user data to delete.

If you believe data may have been collected in error or have questions about your privacy, please contact us at: vrgeek38@gmail.com
